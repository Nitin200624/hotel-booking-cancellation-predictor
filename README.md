# Hotel Booking Cancellation Prediction System

A Machine Learning web application that predicts hotel booking cancellations using Random Forest.

## Features
- Predict booking cancellation risk
- Interactive Streamlit dashboard
- Luxury hotel UI
- Real-time prediction
- Risk visualization

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Random Forest
- Streamlit
- Plotly

## Model Performance
Accuracy: **84.3%**

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py


## Model File Notice
